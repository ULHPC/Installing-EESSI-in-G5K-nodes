# Installing EESSI in G5K nodes

Clone the repository in a shared directory that is accessible to all nodes.

```shell
git clone https://github.com/ULHPC/Installing-EESSI-in-G5K-nodes.git
```

For instance, you can clone the directory in your hope repository. Then, simply execute the installation command:

```
cd Installing-EESSI-in-G5K-nodes
./install-eessi-in-g5k-job-nodes <job ID>
```
